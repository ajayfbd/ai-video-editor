## üöÄ Quick Start

1. **Installation**: `pip install -r requirements.txt`
2. **Setup**: `python -m ai_video_editor.cli.main init`
3. **Configure**: Add your API keys to `.env`
4. **Process**: `python -m ai_video_editor.cli.main process video.mp4`
